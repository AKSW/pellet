Pellet Reasoner Plug-in for Protégé 5.x (fork by ignazio1977)

This plug-in makes OWL-DL reasoner Pellet available in Protégé 5.x. Pellet is a dual-licensed open source Java-based OWL-DL reasoner developed and commercially supported by Complexible Inc. The version available here is a fork from Pellet 2.3.2 that uses OWLAPI 4.0.x

Thanks for using Pellet.

The Pellet Team at Complexible Inc (and the fork maintainer).
